Build a complete static Flutter mobile app called
"WorkSetu - Worker App" — a location-based workforce
dispatch platform. This is the WORKER side only.

==============================================
TECH & GENERAL RULES
==============================================

- Framework: Flutter (Dart)
- All data is hardcoded — NO backend, NO API calls, NO Firebase
- Use Google Fonts — Poppins for all text
- Use only these packages: google_fonts, smooth_page_indicator
- All navigation must work using Flutter Navigator named routes
- Every screen must be a separate Dart file
- App must run on both Android and iOS
